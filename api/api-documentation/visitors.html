API Documentaion



This is the API documentation for the visitors table. It contains the schema and route end points for the visitors table.

visitors end points is located at the following URL: http://your-server-name/visitors

media type:application/json

HTTP methods: GET, POST, PUT, and DELETE

schema
  • schema
routes
  • GET /visitors/all # list all visitors
  • GET /visitors/all/{visitor_id} # get visitors detail
  • GET /visitors/filter?column={column_name}&value={value} # filter visitors by single column
  • GET /visitors/report?{column_name}={value} # report visitors by single/multiple column using & operator
  • POST /visitors/create # create one visitors
  • POST /visitors/create/list # create multiple visitors
  • PUT /visitors/update/{visitor_id} # update visitors by visitor_id
  • PUT /visitors/update?column={column_name}&value={value} # update visitors by any column
  • DELETE /visitors/delete/{visitor_id} # delete one visitors
  • DELETE /visitors/delete # delete multiple visitors
